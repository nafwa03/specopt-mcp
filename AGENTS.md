**🚀 One-Command Developer Quickstart Guide**

Include this brief guide in your portfolio documentation so others can run your entire agentic skill suite in under 60 seconds.

*`# 1. Clone the directory and navigate into it`*  
`cd specopt-mcp`

*`# 2. Spin up and activate a localized Python sandbox environment`*  
`python -m venv venv`  
`source venv/bin/activate  # On Windows use: venv\Scripts\activate`

*`# 3. Install all unified project packages simultaneously`*  
`pip install -r requirements.txt`

*`# 4. Verify system skills pass architectural unit test criteria`*  
`pytest -v`

*`# 5. Launch the Agentic Brain Reasoning Loop`*  
*`# (Make sure LM Studio is running on port 1234 before executing!)`*  
`python agent_brain.py`
